Методы чисел:

  • "папа" всех чисел Number - для преобразования строки в число
    const valueA = "5";
    console.log(Number(valueA)); // 5
  • Number.parseInt() - парсит из строки целое число.
    console.log(Number.parseInt("5px")); // 5
    console.log(Number.parseInt("12qwe74")); // 12
  • Метод Number.parseFloat() парсит из строки дробное число. Метод отвечает на вопрос "Это Not A Number?"
    const validNumber = Number("51"); // 51

Методы массивов:

  • push() - добавляет элемент в конец массива
  • pop() - удаляет и возвращает последний элемент массива
  • shift() - удаляет и возвращает первый элемент массива
  • unshift() - добавляет элемент в начало массива
  • splice() - добавляет или удаляет элементы из массива
  • slice() - возвращает новый массив, содержащий копию части исходного массива
  • sort() - сортирует элементы массива
  • reverse() - меняет порядок элементов массива на обратный
  • map() - применяет функцию к каждому элементу массива и возвращает новый массив
  • filter() - фильтрует элементы массива и возвращает новый массив
  • reduce() - применяет функцию к каждому элементу массива с сохранением промежу
  • concat() - соединяет два или более массива в один новый массив
  • join() - объединяет все элементы массива в одну строку, разделенную заданным разделителем
  • indexOf() - находит индекс первого вхождения элемента в массив, или возвращает -1, если элемент не найден
  • lastIndexOf() - находит индекс последнего вхождения элемента в массив, или возвращает -1, если элемент не найден
  • forEach() - выполняет функцию для каждого элемента массива
  • every() - проверяет, удовлетворяют ли все элементы массива указанному условию
  • some() - проверяет, удовлетворяет ли хотя бы один элемент массива указанному условию
  • find() - находит первый элемент массива, удовлетворяющий указанному условию, или возвращает undefined
  • findIndex() - находит индекс первого элемента массива, удовлетворя
  • flat() - создает новый массив со вложенными массивами в одномерный массив
  • flatMap() - создает новый массив путем использования функции для каждого элемента массива и соединения всех результатов
  • entries() - возвращает итератор пар ключ-значение для каждого элемента массива
  • keys() - возвращает итератор ключей для каждого элемента массива
  • values() - возвращает итератор значений для каждого элемента массива
  • fill() - заполняет массив одним значением начиная с определенного индекса
  • copyWithin() - копирует элементы массива в другую позицию внутри массива
  • includes() - определяет, содержит ли массив определенное значение и возвращает логическое значение.
